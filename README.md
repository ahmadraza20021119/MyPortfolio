## 🌟 Key Features
- **Dynamic Data-Driven Architecture:** All sections (about, journey, projects, skills, certificates) are fed dynamically from a single file (`src/data/portfolioData.js`).
- **Interactive 3D Elements:** Features responsive 3D CSS orbit rings, glowing rotating cubes, and cursor-following canvas space backgrounds.
- **Micro-Animations & Visuals:** Sleek card-tilt effects mapped to cursor movements, interactive custom cursors, and glowing intersection observer transition states.
- **Embedded Document Viewer:** Seamless PDF modal reader directly integrated for immediate credential and paper viewing.
- **Fully Responsive Layout:** Beautifully optimized across mobile, tablet, and widescreen viewports using flexbox and grid layouts.

## 💻 Getting Started Locally

### Prerequisites
Make sure you have **Node.js** and **npm** installed on your system.

### Steps to Run
1. **Clone the Repository:**
   ```bash
   git clone https://github.com/ahmadraza20021119/MyPortfolio.git
   cd MyPortfolio
   ```

2. **Install Dependencies:**
   ```bash
   npm install
   ```

3. **Run Dev Server:**
   ```bash
   npm run dev
   ```

4. **Build for Production:**
   ```bash
   npm run build
   ```
